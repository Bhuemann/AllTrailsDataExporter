# AllTrails Data Exporter
This script extracts all trails and their associated information from Alltrails.com and stores in CSV format.

Data extracted will have the following rows:

- `Unique_Id` 
- `Alt_Id`
- `Trail_Name` 
- `Source` 
- `Distance` 
- `Elevation_Gain` 
- `Highest_Point` 
- `Difficulty` 
- `Est_Hike_Duration` 
- `Trail_Type` 
- `Permits`(Unused)
- `Rating` 
- `Review_Count` 
- `Area` 
- `State` 
- `Country` 
- `Latitude` 
- `Longitude` 
- `Url` 
- `Cover_Photo` 
- `Parsed_Date`
